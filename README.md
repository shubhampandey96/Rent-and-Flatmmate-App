# 🏠 Rent & Flatmate Finder

A full-stack MERN application that helps users discover rental properties, find compatible flatmates, and communicate in real-time. The platform enables property owners to list rooms/apartments while tenants can browse listings, create profiles, express interest, and connect with potential roommates.
